A Raven Monologue

独立/休闲/氛围/放松/试验性/手绘

游戏简介

关于这款游戏

A Raven Monologue是一个简短的实验性无声故事，讲述了一只不知如何叫的乌鸦和他与城里人的关系。

特色:

-沉默的乌鸦
-虚构的人
-手绘艺术
-一个你可以无限解读的故事
-来自Christabel Annora的配乐

系统需求

Windows 7 64 bit/MacOS 10.10.5 64 bit
